## How to Build from Source: (wip)

1. Open the Project with Visual Studio (CPP-Game.sln)
2. Add raylib from the nuget packages
3. Build

- To "pack"/move the Game copy the `assets` folder, and the executable at `%PROJECT%\(x64|Win32|etc)\(Debug|Release)\CPP-Game.exe` to another folder
